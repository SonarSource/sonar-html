The interpolated value is unknown at analysis time and could leak:

  • Version numbers and host names
  • Full, server-side path names
  • Sensitive user data

Static HTML comments are not flagged by this rule, since they cannot carry server-side data into the rendered page. Comments that interpolate server-side values are flagged because the analyzer cannot tell whether the interpolated variable is sensitive.

Ask Yourself Whether

  • The interpolated value could ever contain sensitive information.
  • The comment can be removed or moved to a server-side-only comment syntax.
